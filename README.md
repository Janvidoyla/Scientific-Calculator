# Scientific-Calculator
 A responsive scientific calculator with a modern UI supporting both basic arithmetic and advanced operations like trigonometric functions (sin, cos, tan) and square root.
 
FEATURES-
i) Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
ii) Advanced functions:
Trigonometric functions: sin, cos, tan
Square root calculations
iii) Modern and responsive UI design
iv) Interactive and user-friendly button interface
v) Fully responsive (works on desktop and mobile devices)

TECH STACK-
HTML5 – Structure
CSS3 – Styling and responsiveness
JavaScript – Logic and functionality
